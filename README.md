# SmartPause

**SmartPause** 是一个智能语音控制工具，可以监听麦克风，当你说话时自动暂停视频播放或音乐，停止说话后自动恢复播放。适用于 B站视频或其他网页视频播放场景。

---

## 功能特点

- 🟢 **智能语音检测**：实时监听麦克风音量，判断是否在说话
- ⏸️ **自动暂停**：说话时自动暂停网页视频或音乐
- ▶️ **自动恢复**：静音超过设定延迟后自动恢复播放
- 🌐 **WebSocket 通信**：Python 后端与浏览器 JS 前端实时交互
- 🎨 **彩色日志输出**：Python 控制台输出不同类型事件的彩色日志
- 🔁 **自动重连**：浏览器端断开后 3 秒自动重连

---

## 项目结构

```

SmartPause/
├─ musicAutoPause.py       # Python 后端，监听麦克风并发送 WebSocket 消息
├─ smartpause.user.js      # JS 脚本（油猴脚本），注入视频播放网页，接收控制命令
└─ README.md

```

## 环境要求

- **Python 3.8+**
- **依赖库**：

```bash
pip install sounddevice numpy websockets colorama
```

* **浏览器**：Chrome 或支持油猴脚本的浏览器
* **油猴插件**：Tampermonkey / Greasemonkey

---

## 安装与使用

### 1. 启动 Python 后端

运行

```bash
python musicAutoPause.py
```

* 启动后控制台显示服务启动和日志
* 默认端口：`ws://localhost:8765`
* 可通过修改 `THRESHOLD` 调整音量检测阈值
* 可通过修改 `DELAY` 调整静音恢复延迟（秒）

### 2. 安装浏览器脚本

1. 安装 **Tampermonkey** 插件
2. 新建脚本，复制 `smartpause.user.js` 内容
3. 脚本会自动连接 Python 后端
4. 打开网站视频页面，浏览器标题显示 `【已连麦】` 表示已连接

---

## 配置参数

**Python 后端** (`musicAutoPause.py`)：

| 参数        | 含义             | 默认值  |
| --------- | -------------- | ---- |
| THRESHOLD | 音量阈值，超过判定为说话   | 0.01 |
| DELAY     | 停止说话后恢复播放延迟（秒） | 2    |

**浏览器端** (`smartpause.user.js`)：

* 默认 WebSocket 地址：`ws://localhost:8765`
* 断开自动重连间隔：3 秒
* 可在脚本中自定义日志打印或弹窗通知

---

## 日志说明

Python 后端使用彩色日志区分不同事件：

| 类型 | 颜色    | 示例                    |                 |
| -- | ----- | --------------------- | --------------- |
| 系统 | 蓝色    | `[Z系统] 服务启动           | 阈值=0.01, 延迟=2s` |
| 连接 | 绿色/红色 | `[Z连接] 浏览器已连接 / 已断开`  |                 |
| 语音 | 黄色/青色 | `[Z语音] 检测到说话 / 检测到静音` |                 |

---

## 注意事项

* 浏览器端必须先安装油猴脚本才能接收控制命令
* Python 后端必须先启动，浏览器端才能连接
* 音量阈值可能需要根据环境调整（背景噪音不同）
* 多浏览器或多页面同时连接时，Python 后端会广播给所有客户端
